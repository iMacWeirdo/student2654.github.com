Unit 4: Internal Research Document

This page demonstrates how a script can bypass screen monitoring by using high-frequency updates and focus-polling.

Key Technical Vectors:

  • Temporal Dithering: The canvas overlay updates at 60fps, which often causes "frame-tearing" in low-bandwidth monitoring streams (like ClassWize).
  • Aggressive Polling: Instead of waiting for a "blur" event, this script checks the window state every 10ms.
  • DOM Removal: When focus is lost, the content isn't just hidden; it is removed from the rendering tree.
